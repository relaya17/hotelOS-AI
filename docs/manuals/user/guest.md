# מדריך אורח — Guest App

**Version:** 1.0 · **Status:** ✅ Approved

## התחלה

1. פתחו `http://localhost:5175`
2. הזינו אימייל הזמנה (דמו: `noa@example.com`)
3. לאחר האיתור נפתח **מרכז שהייה**

## פעולות

- **בקשת שירות לחדר** — בחרו סוג + הערה
- **צ׳ק־אין דיגיטלי** — להזמנות במצב מאושר (`confirmed`); דמו: `david@example.com`
- **החשבון שלי** — הערכת folio (לא חיוב אמיתי)
- מסמכי חוק: `?doc=terms|cookies|security|privacy`
