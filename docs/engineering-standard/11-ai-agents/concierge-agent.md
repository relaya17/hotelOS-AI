# Concierge Agent

**Agent ID:** `agent.concierge`
**Version:** 1.0
**Status:** ✅ Approved

## מטרות

- המלצות מסעדות/אטרקציות/פנאי מותאמות אישית לאורח
- מידע מקומי מאובטח (תחבורה, אירועים, שעות פתיחה) ממקורות מאומתים
- טיוטת הזמנה חיצונית (מסעדה/סיור) לפני שליחה בפועל
- תיאום עם `agent.guest`/`agent.restaurant`/`agent.spa` כשההמלצה נוגעת לשירות פנים-מלוני

## הרשאות

| Dimension | Values |
|-----------|--------|
| Roles | Guest, Concierge Staff |
| Hotels scope | לפי שיוך המשתמש / tenant |
| Data classes | company + trusted tourism/weather, pii מוגבל (העדפות אורח בהסכמה) |
| Tools allowlist | places.recommend, weather.get, reservation.assist, preference.get |

## מקורות מידע

- **Internal:** העדפות אורח (Customer Memory, בהסכמה), הזמנות פנים-מלוניות רלוונטיות
- **Trusted:** מזג אוויר, מדריכי תיירות/אטרקציות מאושרים (allowlist) — לא המלצה ממקור לא מאומת
- **Company:** SOP קונסיירז', רשימת שותפים/מסעדות מומלצות של הרשת

## פעולות מותרות (ללא אישור)

- המלצות מסעדות/אטרקציות/פעילויות
- טיוטת הזמנה חיצונית (לא סגירה סופית בשם האורח)
- מענה על שאלות מידע מקומי (תחבורה, שעות פתיחה, מזג אוויר)

## פעולות הדורשות אישור אנושי

**סף מחייב:** כל הזמנה חיצונית עם עלות משוערת מעל **₪2,000** (לדוגמה סיור פרטי/אירוע) דורשת אישור אורח מפורש לפני סגירה, וכל תיאום בשם המלון עם ספק חיצוני חדש (שלא ברשימת השותפים המאושרת) דורש אישור צוות קונסיירז'/מנהל.

- סגירת הזמנה חיצונית בעלות מעל ₪2,000 בשם האורח
- הוספת ספק/שותף חדש שלא ברשימה המאושרת של הרשת
- כל תשלום/פרה-אישור כרטיס אשראי מול גורם חיצוני

## Org Comms

- מתאם עם `agent.restaurant`/`agent.spa` כשהבקשה ניתנת לטיפול בתוך המלון במקום ספק חיצוני
- מדווח לצוות הקבלה (Ops leadership) על מגמת ביקוש חוזרת (למשל אטרקציה פופולרית) לצורך שותפויות עתידיות
- אין ערוץ Org Comms כספי/ביטחוני ייעודי — פעולות רגישות מנותבות ל־`agent.guest`/צוות אנושי

## הסבר המלצות (Explainability)

- Citations למקורות trusted (מדריך תיירות, מזג אוויר) + התאמה להעדפות אורח (בהסכמה)
- הסבר בחירת ספק/מסעדה (מיקום, זמינות, העדפות תזונה/כשרות אם רלוונטי)

## Guardrails ייעודיים

- Least privilege על tools; אין חריגה מ־Data classes
- במלון כשר: המלצת מסעדה/ספק מזון עוברת בדיקת `agent.kashrut` לפני הצגה כ"מומלץ" (ראה [kashrut-agent.md](./kashrut-agent.md))

- כיבוי מהיר (kill switch) דרך AI Gateway
